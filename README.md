# מערכת תוכניות עבודה · כוחות עזר — הרצליה מדיקל סנטר

אפליקציית React (Vite) בעברית מלאה (RTL) לניהול תוכניות עבודה וניקיון של כוחות העזר.

## מבנה
- **מסך ראשי** — כל המחלקות והיחידות ככרטיסים.
- **קבוצות נפתחות בעץ הניווט:**
  - **מחלקות אשפוז** (3 משמרות: בוקר / ערב / לילה): אשפוז א, אשפוז ב, אשפוז ג, טיפול נמרץ, אשפוז ה, אשפוז ד.
  - **מכון דימות** (3 משמרות): CT, MRI, רנטגן.
- **יחידות עצמאיות** (2 משמרות: בוקר / ערב): התאוששות, חדרי ניתוח, IVF, צינתורים, גסטרו ופעולות.
- בכל יחידה **לוח חלונות**. חלון פעיל: **תוכניות ניקיון מחלקתי** (יומית / שבועית / חודשית, צ׳קליסט, בחירת משמרת אוטומטית לפי שעה, שם מבצע חובה, נעילה לקריאה בלבד לאחר שמירה).

## הרצה
```bash
npm install
npm run dev
```
לאחר מכן ייפתח הדפדפן בכתובת http://localhost:5173

## שפת עיצוב
- **רקע**: גרדיאנט כחול-תכלת עמוק (#0A4D8C ל-#17A2B8) עם כתמי אור עדינים.
- **פאנלים**: גלאסמורפיזם — רקע שקוף, `backdrop-filter: blur`, מסגרת בהירה וצל רך.
- **כפתורים**: מעוגלים (pill), בעלי נפח — צל וגרדיאנט מותג, אפקט הרמה ב-hover.
- **גופן**: Heebo (נטען מ-Google Fonts), RTL מלא.
- **אייקונים**: סט קווי ניטרלי ומכובד לסביבה רפואית בלבד — ללא אמוג'י דרמטיים.
- **רספונסיב**: מותאם מובייל (סרגל הצד עובר לראש המסך מתחת ל-900px).

## תוכן הניקיון
תוכניות הניקיון האמיתיות מקודדות ב-`src/data/cleaningTemplates.js` ישירות מקובצי
ה-Word שבתיקיית `cleaning programs`. לכל יחידה מבנה משלה (משמרות / תחנות / חדרים),
רשימות "בין מטופלים" כמונה מהיר ללא חתימה, ובלוקי סגירה עם שדות חתימה חובה
(שם מבצע + חתימת ביצוע + נבדק עיי חתימה) ונעילה לקריאה בלבד. הארכיון נשמר בעברית בלבד.

## טכנולוגיה
React 18 · Vite 5 · ללא תלות חיצונית בעיצוב (CSS גלאסמורפיזם בצבעי כחול-תכלת).
